# SmartHomeMiniApp
  - Mini app for UI component
  - SwiftUI
    
## Installation
1. Clone the repository: `https://github.com/Jamorno/SmartHomeMiniApp.git`
2. Open the project in Xcode.
3. Run the app on a simulator or device.
